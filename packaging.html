Uplifting a Package

Fantasia is a well known chips brand in Lebanon. The concept of this project is to uplift the chips packaging and make it more user friendly. The package was inspired by the grip of the hand, the edged package makes it easier for the user to hold the package in hand firmly. The package also introduced the option of storage, were the user can cover the package easily for later usage.
